# V1 QA 보고서

검토일: 2026-07-12  
범위: 바울의 1·2·3차 전도여행, 로마 압송

## 구현 검토

- 정적 사이트 구조로 GitHub Pages 루트와 하위 프로젝트 경로에서 상대 URL을 사용한다.
- 지도 확대·축소·드래그·핀치 줌, 장소 선택, 여정 필터, 검색, 자동 재생, 이전·다음, 속도 조절을 구현했다.
- 데스크톱은 우측 패널, 모바일은 하단 시트로 상세 정보를 표시한다.
- 키보드 포커스, skip link, 버튼 레이블, 대화상자, 최소 터치 크기와 reduced-motion 설정을 포함한다.
- 서비스 워커는 로컬 리소스만 캐시하고 OpenStreetMap 타일은 캐시하지 않는다.

## 데이터 검토

- 장소 54개
- 방문 기록 70개
- GeoJSON 이동 선분 66개
- 대표 사건 15개
- 모든 stop에 방문 순서, 이전/다음 stop, 동행자, 장절, 사건 요약, 신뢰도 기록
- 동일 도시의 재방문은 별도 stop으로 보존

## 본문 대조 핵심

- 1차: Acts 13:1–14:28의 출발, 구브로, 버가, 남부 소아시아, 역순 귀환을 반영했다.
- 2차: Acts 15:36–18:22의 실라 동행, 디모데 합류, 마게도냐 이동, 아가야 사역, 에베소·가이사랴 귀환을 반영했다.
- 3차: Acts 18:23–21:17의 에베소 장기 사역, 마게도냐·헬라, 드로아와 해상 귀환 경로를 반영했다.
- 로마 압송: Acts 27–28의 시돈, 구브로 아래 항로, 무라 환승, 크레타, 파선, 멜리데, 이탈리아 육로를 반영했다.

## 의도적으로 확정하지 않은 항목

- Acts 18:22의 교회는 UI에서 예루살렘으로 제시하지만 `inferred`로 표시한다.
- 3차 여행의 헬라 석 달 체류는 고린도 단일 도시가 아니라 아가야 지역으로 표시한다.
- 더베·뵈닉스·삼관 좌표는 후보 또는 근사 지점이다.
- 로마 압송의 크레타–멜리데 선은 폭풍 속 실제 항로 복원이 아닌 시각적 연결이다.
- 여행 연대는 절대 확정값이 아니므로 “약”과 학자 간 차이 문구를 사용한다.

## 배포 전 수동 확인 권장

- Chrome, Edge, Safari iOS, Android Chrome에서 지도 터치와 하단 시트 확인
- GitHub Pages 실제 하위 경로에서 서비스 워커와 JSON 상대 경로 확인
- 공개 트래픽 규모에 맞는 지도 타일 제공자 검토
- 사용하려는 한국어 성경 번역본의 링크·본문 인용 권한 검토
